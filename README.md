# College Football Fourth Down Data Story

This repository is for the data analysis conducted for a data-driven journalistic piece exploring fourth down trends in college football between 2016 and 2025. This project was completed for the Northwestern data journalism course JOUR 377 in the winter of 2026.

## What's in this Repo?

In this repository, you can find all my subfolders/directories, and all the documents I worked in to complete my data story.

### Subfolders/Directories

- [`data/`](data): contains data for this project
- [`plots/`](plots): contains png files for the visualizations I created for my story

### R Scripts

- `1-load-clean-data.R`: document for downloading data from API and filtering it to only include fourth down plays
- `2-process-analyze-viz.R`: document for continued processing of data, as well as analysis and visualization building

### Reports

- `story-pitch.html`: rendered html version of story memo
- `story-pitch.qmd`: file for crafting story memo
